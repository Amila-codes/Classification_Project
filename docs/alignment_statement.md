## Alignment Statement

The dataset selected for this project is an ISP customer churn dataset sourced from Kaggle. It contains structured customer-level data related to internet service usage, billing, and subscription details. Key features include variables such as subscription_age, bill_avg, remaining_contract, service_failure_count, download_avg, and upload_avg, along with indicators like is_tv_subscriber and is_movie_package_subscriber. This dataset is directly aligned with the classification task of predicting whether a customer will churn.

The target variable is churn, a binary label indicating whether a customer has left the service (1) or remained (0). These classes are already defined, so no relabeling is required. From a technical perspective, the dataset includes meaningful behavioral and account-based features that are assumed to be available at the time of prediction. From an ethical standpoint, the dataset does not include explicit sensitive attributes such as gender or race, and it is released under a CC0 license on Kaggle, allowing unrestricted use. Some limitations remain, including moderate class imbalance and the absence of explicit time-based features, so predictions are interpreted as churn risk rather than a strictly time-bound forecast.
