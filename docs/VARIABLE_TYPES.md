## Variable Types Documentation

This file describes the features used in the ISP Customer Churn dataset and their corresponding data types. It helps guide preprocessing steps and ensures consistency in model development.

- Binary: Categorical variables with two possible values (e.g., 0 = No, 1 = Yes)
