# Data Dictionary

## Metadata Summary

The ISP dataset contains customer-level data used to predict churn. The target variable, Churn, is a binary label indicating whether a customer has left the service. The dataset includes features related to customer account information, service usage, billing behavior, and subscription details. Each row represents a single customer, and all features are structured for supervised classification.

The dataset consists primarily of structured tabular data with numerical features. There is no image or sensor-based metadata, and no complex annotation protocol beyond binary labeling. All features are assumed to be recorded prior to the prediction point, ensuring no data leakage. The dataset does not include explicit time-based features. However, all variables are assumed to reflect historical customer behavior available at the prediction point.
